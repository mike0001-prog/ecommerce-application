# Customizing Bootstrap with Your Cyberpunk Color Scheme

To make your custom cyberpunk colors take precedence over Bootstrap's default colors, you'll need to override Bootstrap's variables and components. Here's how to do it while maintaining your futuristic aesthetic:

## 1. First, include Bootstrap in your head section (before your custom CSS):

```html
<!-- Bootstrap CSS -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
```

## 2. Create a custom CSS file (or style section) that overrides Bootstrap:

```html
<style>
  :root {
    --black: #000000;
    --cyan: #00FFEE;
    --pink: #FF00FF;
    --white: #FFFFFF;
    --gray: #1A1A1A;
    
    /* Override Bootstrap variables */
    --bs-body-bg: var(--black);
    --bs-body-color: var(--white);
    --bs-primary: var(--cyan);
    --bs-secondary: var(--pink);
    --bs-border-color: var(--cyan);
  }

  /* Apply your custom styles after Bootstrap */
  body {
    font-family: 'Roboto', sans-serif;
    background-color: var(--black);
    color: var(--white);
  }

  /* Navbar customization */
  .navbar {
    border-bottom: 1px solid var(--cyan);
  }

  .navbar-dark .navbar-nav .nav-link {
    color: var(--white);
    transition: 0.3s;
  }

  .navbar-dark .navbar-nav .nav-link:hover {
    color: var(--cyan);
    text-shadow: 0 0 10px var(--cyan);
  }

  /* Buttons */
  .btn-neon {
    background-color: transparent;
    color: var(--cyan);
    border: 2px solid var(--cyan);
    padding: 10px 20px;
    font-family: 'Orbitron', sans-serif;
    transition: 0.3s;
    box-shadow: 0 0 10px var(--cyan);
  }

  .btn-neon:hover {
    background-color: var(--cyan);
    color: var(--black);
    box-shadow: 0 0 20px var(--cyan);
  }

  .btn-neon-pink {
    background-color: transparent;
    color: var(--pink);
    border: 2px solid var(--pink);
    padding: 10px 20px;
    font-family: 'Orbitron', sans-serif;
    transition: 0.3s;
    box-shadow: 0 0 10px var(--pink);
  }

  .btn-neon-pink:hover {
    background-color: var(--pink);
    color: var(--black);
    box-shadow: 0 0 20px var(--pink);
  }

  /* Custom neon effects */
  .neon-cyan {
    color: var(--cyan);
    text-shadow: 0 0 10px var(--cyan);
  }

  .neon-pink {
    color: var(--pink);
    text-shadow: 0 0 10px var(--pink);
  }

  /* Product cards */
  .product-card {
    background-color: var(--gray);
    border: 1px solid var(--cyan);
    transition: 0.3s;
  }

  .product-card:hover {
    transform: translateY(-10px);
    box-shadow: 0 0 20px var(--cyan);
  }
</style>
```

## Key Points for Overriding Bootstrap:

1. **CSS Specificity**: Your custom CSS should come after Bootstrap's CSS
2. **Variable Overrides**: Redefine Bootstrap's CSS variables with your colors
3. **Component Customization**: Target Bootstrap components with your selectors
4. **Responsive Preservation**: Keep Bootstrap's responsive utilities intact
5. **Custom Classes**: Maintain your special effects (neon glows, etc.) with custom classes

## Additional Customizations You Might Want:

```css
/* Custom dropdown to match your theme */
.dropdown-menu {
  background-color: var(--gray);
  border: 1px solid var(--cyan);
}

.dropdown-item {
  color: var(--white);
}

.dropdown-item:hover {
  background-color: var(--black);
  color: var(--cyan);
}

/* Custom form controls */
.form-control {
  background-color: var(--gray);
  border-color: var(--cyan);
  color: var(--white);
}

.form-control:focus {
  background-color: var(--gray);
  color: var(--white);
  border-color: var(--cyan);
  box-shadow: 0 0 10px var(--cyan);
}
```

This approach gives you Bootstrap's layout and responsive capabilities while maintaining your distinctive cyberpunk visual style.
